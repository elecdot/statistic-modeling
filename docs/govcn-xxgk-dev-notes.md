# gov.cn XXGK Development Notes

This document records source-specific development notes and current artifact
snapshots for the central `gov.cn/zhengce/xxgk` corpus. Keep implementation
details in `docs/govcn-xxgk-crawler.md` and field definitions in
`docs/data-dictionary.md`.

  semantics, pagination, and compliance notes are documented in notebooks
  `20_central_gov_xxgk_explore` and `20a_decision`.

`30_central_gov_xxgk_corpus_qa.ipynb`.

  `just govcn-xxgk-processed-v0` and registered in the source manifest.

## Stage Decisions

- `official_subject_categories` is preserved as a gov.cn source-provided
  taxonomy. It should not be treated as a province-level policy-tool taxonomy
  without a separate mapping decision.
- Processed v0 keeps only reproducible `attachment_urls` in the main corpus.
  Exploratory attachment context outputs remain outside the processed schema.
- Attachment context may not be needed for this project. If needed later, design
  it as a separate artifact rather than adding exploratory fields to the main
  processed corpus.
