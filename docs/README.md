# Docs

Research notes, reports, and project documentation live here.

## Core Documents

- `govcn-xxgk-crawler.md`: technical handoff for the gov.cn XXGK crawler, including commands, run mechanism, output schema, and quality checks.
- `govcn-xxgk-dev-notes.md`: source-specific development notes, milestone history, current artifact snapshot, and stage decisions for gov.cn XXGK.
- `manual-srdi-policy-mining-notes.md`: milestone and paper-drafting notes for the manually collected SRDI policy-mining workflow.
- `source-manifest-guide.md`: rules for registering raw and derived data artifacts in `data/source-manifest.csv`.
- `data-dictionary.md`: field definitions for registered data artifacts.

Keep docs aligned when crawler configs, output schemas, or manifest conventions change.
